# Política de Segurança

## Versões com suporte

Como este projeto é um portfólio estático, a versão com suporte é sempre a branch `main`.

## Reportando uma vulnerabilidade

Se você identificar uma vulnerabilidade de segurança, não abra uma issue pública.

Envie os detalhes para:

- Email: [leoodonvito@gmail.com](mailto:leoodonvito@gmail.com)

Inclua, se possível:

- descrição do problema
- impacto potencial
- passos para reproduzir
- evidências ou capturas de tela
- sugestão de correção, se houver

O objetivo é confirmar o recebimento rapidamente e tratar o caso com prioridade razoável.

## Boas práticas deste repositório

- Não armazenar segredos ou credenciais no código.
- Revisar links, embeds e scripts de terceiros antes de publicar.
- Validar alterações em SEO, metadados e recursos estáticos antes do deploy.
